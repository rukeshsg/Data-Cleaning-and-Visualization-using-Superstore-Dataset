## 📊 Project Overview
This project focuses on data cleaning and visualization using the Superstore dataset. It includes data preprocessing, analysis, and an interactive dashboard built using Streamlit to display key insights.

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Streamlit  

---

## 📁 Project Structure
- Dataset – Raw and cleaned data  
- Notebook – Data cleaning and analysis  
- Dashboard – Streamlit app  
- Report – Final project report  
- Images – Visualizations  

---

## 📈 Features
- Data cleaning and preprocessing  
- Sales and profit analysis  
- Multiple visualizations  
- Interactive dashboard with filters
